DPS
What is DPS?
DPS, short for dcv's plug.dj script, is a bunch of javascript written to improve your time on plug.dj. It consists of some pretty cool features, and the whole script is still in the works.
DPS RELEASE
DPS RELEASE is a non buggy version ready for user use!
DPS BETA
DPS BETA is a constantly changing version of DPS, you can get a sneak peak of all the features here, even though they might not work just yet.
